# WebGIS 前端学习路线

> 本文件概述学习阶段、目标与时间预估，后续会在每个阶段完成后更新实际耗时。

## 阶段 1 – GIS 基础概念
- 目标：理解坐标系、投影、GeoJSON、WMS/WFS
- 预计时间：1 周

## 阶段 2 – 前端制图库入门（Leaflet / OpenLayers）
- 目标：在页面渲染矢量与栅格图层，实现平移、缩放、标记
- 预计时间：2 周

## 阶段 3 – 高级交互与工具
- 目标：图层分组、弹窗、测距/测面积、空间查询
- 预计时间：1 周

## 阶段 4 – 3D 与 WebGL（Cesium / Mapbox GL）
- 目标：创建倾斜/三维视图、点云展示
- 预计时间：2 周

## 阶段 5 – 数据处理与性能优化
- 目标：瓦片切片、向量瓦片、服务端 WMS、GeoServer
- 预计时间：2 周

## 阶段 6 – 项目实战（综合项目）
- 目标：完整前端 GIS 应用，包含登录、权限、后端 API 对接
- 预计时间：3 周

## 阶段 7 – 部署与 CI
- 目标：使用 Vite/webpack 打包，部署到 GitHub Pages 或 Netlify，配置自动化测试
- 预计时间：1 周
