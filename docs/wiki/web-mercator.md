# Web Mercator

Web Mercator（网页墨卡托投影）是一种广泛用于网络地图服务的圆柱投影方式，是传统墨卡托投影的改进版本。它被设计为在Google Maps、Bing Maps、OpenStreetMap等在线平台上快速渲染瓦片地图的标准坐标系统。

## 关键事实

- 类型：伪墨卡托圆柱投影

- EPSG代码：3857（或早期使用900913）

- 基准椭球体：WGS 84

- 用途：网络地图瓦片渲染与坐标匹配

- 特点：保角但不保面积，高纬度地区夸张严重

## 投影原理与特点

Web Mercator 与传统墨卡托投影相似，都通过等角投影保持方向角不变，使地图在局部区域内角度精度高、形状失真小。然而它并未使用真实的地球椭球体，而是假设地球为完美球体。此简化使得计算经纬度与平面坐标间转换更高效，但会在高纬度地区显著夸大面积。

## 技术应用

该投影是网络地图瓦片（tile map）的默认标准坐标系。由于瓦片以固定像素分辨率排列，Web Mercator 的等角特性便于在多缩放级别间无缝拼接图像和叠加矢量数据。多数GIS与地图API（如ArcGIS Online）均采用此系统。

## 局限与争议

尽管实用性强，Web Mercator在测量、面积计算和高纬度制图中误差较大，不适合精确地理分析。一些地理学者批评其“视觉偏见”，因为极地区域被不成比例地放大，但其简洁计算方式仍使其成为在线地图的事实标准。
